## 介绍

![logo](public/logo.png)

PerfEdge 是一个专注于前端性能优化的知识库，旨在帮助开发者更好地理解和使用前端性能优化技术，提高网站和应用的用户体验。
知识库特色：

1. 可体检交互：不同于传统知识库，本知识库设计了很多可交互组件，让大家尽可能在使用时体验到前端性能瓶颈。
2. 持续更新：由一群爱好者发起，持续更新。
